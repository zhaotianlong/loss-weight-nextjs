# 减肥训练营管理系统 (Weight Loss Camp Management System)

一个完整的企业级减肥训练营（减肥营）管理后台系统，支持多营地、多角色管理，包含学员管理、课程安排、菜谱制定、费用收取等完整功能。

## 🎯 系统概览

### 核心模块
- **👥 学员管理**：学员全生命周期管理、身体数据追踪、入住/换房/离营流程。
- **📚 课程管理**：公共课程表、私教课预约与进度追踪。
- **🍽️ 今日安排**：集成每日排班与餐饮推荐，提供营养分析。
- **💰 财务管理**：学费收取、续费预警、业绩指标统计。
- **📊 仪表盘**：多营地数据看板、核心 KPI 统计、待办事项提醒。

## 🛠️ 技术栈
- **Frontend**: Next.js 15+ (App Router), React 19, TypeScript, Ant Design 5.x, Tailwind CSS.
- **Backend**: Next.js API Routes, JWT Auth.
- **Mocking**: MSW (Mock Service Worker) & Persistent Mock Data Layer.

## 🚀 快速开始

1. **安装依赖**:
   ```bash
   npm install
   ```

2. **启动开发服务器**:
   ```bash
   npm run dev
   ```

3. **访问系统**:
   打开 [http://localhost:3000](http://localhost:3000)

# 减肥训练营管理系统

一个完整的企业级减肥训练营管理后台系统，基于 Next.js 16 + React 19 + TypeScript + Tailwind CSS v4 构建。

## 🎯 项目概述

这是一个专业的减肥训练营管理解决方案，提供：
- ✅ 多营地、多角色管理系统
- ✅ 完整的学员生命周期管理
- ✅ 课程和菜谱管理
- ✅ 费用收取与续费管理
- ✅ 数据分析与报表统计
- ✅ 13 个详细的 Mermaid 系统架构图

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看应用。

### 访问仪表板
- 开发地址：[http://localhost:3000/dashboard](http://localhost:3000/dashboard)

## 📚 核心功能模块

| 模块 | 描述 | 相关页面 |
|------|------|--------|
| **👥 学员管理** | 学员信息、身体数据追踪、入住管理 | `/dashboard/students` |
| **📚 课程管理** | 公共课程（免费）、私教课程（付费） | `/dashboard/courses` |
| **🍽️ 菜谱管理** | 菜谱库、每日菜单、营养分析 | `/dashboard/recipes` |
| **💰 费用管理** | 学费收取、续费管理、收费标准维护 | `/dashboard/tuition` |
| **🏕️ 营地管理** | 多营地支持、房间管理、床位分配 | `/dashboard/camps` |
| **👔 员工管理** | 员工信息、角色权限分配 | `/dashboard/staff` |
| **📊 报表统计** | 学员统计、营地统计、财务统计 | `/dashboard/reports` |

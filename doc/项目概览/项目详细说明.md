# 减肥训练营管理系统

一个完整的企业级减肥训练营（减肥营）管理后台系统，支持多营地、多角色管理，包含学员管理、课程安排、菜谱制定、费用收取等完整功能。

## 🎯 系统概述

### 核心模块

- **👥 学员管理**：学员信息、身体数据追踪、入住管理、进度报告
- **📚 课程管理**：公共课程（免费）、私教课程（付费）、课程表安排
- **🍽️ 菜谱管理**：菜谱库、每日菜单、营养分析
- **💰 费用管理**：学费收取、续费管理、营地收费标准维护
- **🏕️ 营地管理**：多营地支持、房间管理、床位分配
- **👔 员工管理**：多角色权限控制（管理员、经理、教练、厨师、保洁、前台）
- **📊 报表统计**：学员统计、营地统计、财务统计、课程分析

### 用户角色

| 角色 | 权限 | 主要职能 |
|------|------|--------|
| **Admin** 管理员 | 全部 | 系统管理、用户管理、多营地管理 |
| **Manager** 经理 | 营地管理 | 营地运营、员工管理、报表查看 |
| **Coach** 教练 | 课程管理 | 课程安排、学员进度追踪、私教管理 |
| **Chef** 厨师 | 菜谱管理 | 菜谱制作、每日菜单安排 |
| **Cleaner** 保洁 | 卫生管理 | 卫生巡查、房间状态维护 |
| **Receptionist** 前台 | 运营支持 | 学员入住、费用收取、课程报名 |

## 🛠️ 技术栈

### 前端
- **Next.js 16** - React 框架
- **React 19** - UI 库

- **Tailwind CSS v4** - 样式框架

- **Next.js API Routes** - RESTful API

- **PostgreSQL** - 主数据库（推荐）

### 部署
- **Vercel** - 前端部署（推荐）

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn
- PostgreSQL 12+（可选，生产推荐）

### 安装依赖

```bash
npm install
```

### 环境变量配置

创建 `.env.local` 文件：

```env
# 数据库
DATABASE_URL=postgresql://user:password@localhost:5432/camp_db

# 认证
JWT_SECRET=your-secret-key-here
NEXT_PUBLIC_API_URL=http://localhost:3000/api

# 支付（可选）
ALIPAY_APPID=xxx
WECHAT_PAY_KEY=xxx

# 文件存储（可选）
AWS_S3_BUCKET=your-bucket
AWS_S3_REGION=us-east-1
```

### 启动开发服务器

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)

## 🎨 核心功能示例

### 1. 仪表板
- 学员统计（总数、在训、已结业等）
- 营地统计（床位占用率、收入等）
- 待续费学员提醒
- 今日菜单展示
- 最近入住记录

### 2. 学员管理
- 学员注册和信息管理
- 身体数据记录和追踪（体重、BMI、体脂率等）
- 入住/退住办理
- 进度报告生成

### 3. 课程管理
- **公共课程**：免费，所有学员可参加，固定时间表
  - 瑜伽、有氧舞蹈、力量训练等
  - 每日固定课程表
  - 参加人数统计
  
- **私教课程**：付费，个性化课程
  - 选择教练和课程类型
  - 购买课时和支付
  - 预约课程时间
  - 课时完成度追踪

### 4. 菜谱管理
- 菜谱库管理（成分、卡路里、宏观营养）
- 每日菜单安排
- 营养分析和热量统计
- 与学员无直接关系，由厨师统一管理

### 5. 费用管理
- **学费收取**：首期费用收取、支付状态跟踪
- **续费管理**：到期提醒、续费处理、新的过期日期计算
- **收费标准**：营地可定义多种套餐（如30天套餐、60天套餐等）
- **支付集成**：支持支付宝、微信支付、银行卡

### 6. 营地管理
- 多营地支持
- 房间管理（楼层、房间号、床位数）
- 床位分配和状态跟踪
- 设施维护记录

### 7. 员工管理
- 员工信息维护
- 角色和权限分配
- 入职/离职管理
- 员工活动日志

### 8. 报表统计
- **学员报表**：人数统计、身体数据分析、进度报告
- **营地报表**：床位利用率、入住统计、收入统计
- **财务报表**：收入汇总、待收费用、续费率
- **课程报表**：课程参加率、私教销售情况、教练评分

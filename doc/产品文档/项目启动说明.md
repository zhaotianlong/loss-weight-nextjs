## 启动项目计划

### 步骤 1：检查并安装依赖
- 检查 `node_modules` 目录是否存在
- 如果不存在，运行 `npm install` 安装项目依赖

### 步骤 2：启动开发服务器
- 运行 `npm run dev` 启动 Next.js 开发服务器
- 服务器将在 `http://localhost:3000` 启动

### 步骤 3：验证项目运行
- 检查服务器启动日志
- 确认没有错误信息
- 提供访问地址和测试账号

### 预期结果
- 开发服务器成功启动
- 可以通过浏览器访问 `http://localhost:3000`
- 可以使用测试账号登录系统
